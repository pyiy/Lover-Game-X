# 情侣飞行棋

一款浪漫有趣的双人飞行棋游戏，支持多端同步功能。

## 功能特点

- 丰富的格子类型：真心话、大冒险、亲亲、抱抱、前进、后退等
- 可自定义格子内容和特殊格子位置
- 游戏计时器，支持快速选择预设时间
- 移动端适配，支持手机浏览器
- 多端同步功能（可选）

## 多端同步配置

如果需要启用多端同步功能，请配置 MySQL 数据库：

### 1. 创建数据库

\`\`\`sql
CREATE DATABASE couple_game CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
\`\`\`

### 2. 配置环境变量

在项目根目录创建 \`.env.local\` 文件：

\`\`\`env
MYSQL_HOST=你的数据库地址
MYSQL_PORT=3306
MYSQL_USER=数据库用户名
MYSQL_PASSWORD=数据库密码
MYSQL_DATABASE=couple_game
\`\`\`

### 3. 启动应用

应用启动后会自动检测数据库配置：
- 如果配置正确，将显示房间管理界面，可以创建或加入房间
- 如果未配置或配置错误，将使用本地单机模式

## 使用说明

1. **创建房间**：点击"创建房间"按钮，系统会生成6位房间号
2. **分享房间号**：将房间号分享给另一位玩家
3. **加入房间**：输入房间号即可加入同一游戏
4. **本地游戏**：也可以选择"本地双人游戏"，两人共用一台设备

## 自定义内容

点击右上角"自定义"按钮可以：
- 编辑各类格子的内容
- 设置特殊格子的固定位置
- 自定义终点奖励
- 导入/导出配置文件
